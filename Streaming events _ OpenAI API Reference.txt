    Streaming events

When you create a Response <https://platform.openai.com/docs/api-
reference/responses/create> with |stream| set to |true|, the server will
emit server-sent events to the client as the Response is generated. This
section contains the events that are emitted by the server.

Learn more about streaming responses <https://platform.openai.com/docs/
guides/streaming-responses?api-mode=responses>.
